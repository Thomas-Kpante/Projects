Project Title: Funko Pop Shop Management System

Description:
The Funko Pop Shop Management System is a web-based application designed to manage user accounts, orders, and products for an e-commerce store specializing in Funko Pop collectibles. The system allows administrators to perform various tasks such as adding, editing, and deleting user accounts, managing orders, and updating the product inventory.

Features:
1. User Account Management:
   - Create new user accounts with different roles (e.g., admin, user).
   - View and edit existing user accounts.
   - Delete user accounts.

2. Order Management:
   - View all orders placed by users.
   - Update order status (e.g., pending, processing, shipped).
   - Delete orders.

3. Product Management:
   - Add new products to the inventory.
   - Edit existing product details.
   - Delete products.

4. Security:
   - Authentication system for user login.
   - Role-based access control to restrict certain functionalities to administrators only.

Installation:
1. Clone the repository to your local machine.
2. Import the provided SQL file to create the necessary database schema and sample data.
3. Configure the database connection settings in the "includes/db_connect.php" file.
4. Host the application on a web server with PHP support (e.g., Apache, Nginx).
